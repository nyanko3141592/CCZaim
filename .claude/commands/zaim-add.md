Zaimに新しい取引を登録します。

## 使い方

ユーザーは自然言語で支出・収入を伝えます:
- 「今日ランチで1200円使った」
- 「スタバでコーヒー550円」
- 「3/15に本を2000円で買った」
- 「給料30万円入った」

## 手順

1. ユーザーの入力から以下を抽出:
   - 種別: 支出 / 収入 / 振替
   - 金額
   - 日付（省略時は今日）
   - カテゴリ・ジャンル（推定）
   - 場所・メモ

2. カテゴリが不明な場合、`python zaim_client.py categories` で一覧を取得して最適なものを選択

3. 登録内容を確認表示:

```
以下の内容で登録します:

| 項目 | 内容 |
|---|---|
| 種別 | 支出 |
| 金額 | ¥1,200 |
| 日付 | 2024-03-20 |
| カテゴリ | 食費 |
| ジャンル | 外食 |
| 場所 | スタバ |

よろしいですか？
```

4. 承認後、zaim_client.py を使って登録:

```python
import zaim_client
s = zaim_client.get_session()
zaim_client.create_payment(s, category_id=..., genre_id=..., amount=1200, date="2024-03-20", place="スタバ")
```

## 注意
- 必ず登録前にユーザーに確認する
- 金額が大きい場合（10万円以上）は特に確認を丁寧に
