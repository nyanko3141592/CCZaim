今月の家計ダッシュボードを表示してください。

## 手順

1. `python zaim_client.py dashboard` を実行してJSONデータを取得
2. 結果を以下のフォーマットで見やすく整形して表示:

### 表示フォーマット

```
## {YYYY年M月} 家計ダッシュボード

### 収支サマリー
| 項目 | 金額 |
|---|---|
| 収入 | ¥XXX,XXX |
| 支出 | ¥XXX,XXX |
| 差引 | ¥±XXX,XXX |

### カテゴリ別支出 TOP10
（棒グラフ風にUnicodeブロック文字で可視化）

### 今月の高額支出 TOP5
（日付・金額・カテゴリ・メモを表形式で）

### 日別支出推移
（簡易グラフで表示）

### 未分類
未分類がX件あります。`/zaim-categorize` で分類できます。
```

3. 特に気になるポイント（前月比で急増したカテゴリなど）があればコメントを添える

## 注意
- 金額は3桁区切りで表示
- ネガティブな差引はそのまま表示（赤字であることを明記）
