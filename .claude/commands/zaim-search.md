Zaimの取引データを検索します。

ユーザーの指示に基づいて `zaim_client.py` を使い、適切な取引を検索・表示してください。

## 使い方

ユーザーは以下のような指示をします:
- 「先月のコンビニ支出を見せて」
- 「3月の食費を調べて」
- 「今月1万円以上の支出」
- 「直近の外食」

## 手順

1. ユーザーの指示から、期間・カテゴリ・金額などの条件を読み取る
2. `python zaim_client.py money --start YYYY-MM-DD --end YYYY-MM-DD --all` で取引データを取得
3. 取得したデータからPythonスクリプトで条件に合う取引をフィルタ
4. 結果を見やすい表形式で表示
5. 合計金額と件数のサマリーを添える

## フィルタ例

```python
import json, sys

data = json.load(sys.stdin)
# カテゴリ名やplace, commentで部分一致フィルタ
# amount で金額フィルタ
```
